# Gartix.sh
Gartic style game based on algorithmetic command-driven drawing.

## Idea
- The game focuses on a hacker-style gameplay of gartic. For example, instead of drawing things with the mouse, you type "draw square 20 20" to draw a square in the canvas, which is small for enhance future ascii rendering.

## Technology
- Server: Node.js with express. I would do it in C or C++, but for servers is unsafe.
- Client: Web and TUI. The client could be on the web browser or on the terminal.
